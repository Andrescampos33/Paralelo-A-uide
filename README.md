# Paralelo-A-uide README Juego del Ahorcado
Software: Juego del Ahorcado

Objetivo: Desarrollar habilidades fundamentales de pensamiento lógico y programación, aplicando conceptos básicos del lenguaje en un problema real y completo.

Introducción al Programa:
El juego del ahorcado es un programa interactivo de carácter educativo que consiste en adivinar una palabra oculta a partir de letras ingresadas por el usuario, contando con un número limitado de intentos. El programa se realiza mediante el uso de un lenguaje de programación, en este caso Python, utilizando estructuras básicas como variables, condicionales, bucles, listas y funciones. A través de estas herramientas, el sistema valida las entradas del usuario, actualiza el estado del juego y muestra el progreso de manera dinámica, permitiendo determinar si el jugador gana o pierde según sus decisiones y el beneficio de realizar este tipo de ejercicios radica en el fortalecimiento del pensamiento lógico, la resolución de problemas y la aplicación práctica de los conocimientos teóricos adquiridos. Además, fomenta la creatividad, mejora la comprensión de la interacción entre el usuario y el programa, y nos prepara como estudiantes para enfrentar proyectos de mayor complejidad en etapas posteriores de su formación académica.
